### 1. 프로젝트내용
+ 프로젝트는 로그인 관련 API이고 세션을 통해 로그인 상태를 유지할 수 있도록 기능을 구현했다.
### 2. 사용된 기술 정리
+ MySQL을 연동해서 사용했고 로그인을 유지시키기 위해 session모듈을 사용했다. 이 둘을 연동시키기 위해 express-mysql-session, mysql 등의 모듈을 사용했다.
### 3. 새로 알게 된 내용
+ 로그인상태를 유지시키기 위한 세션, 쿠키에 대해 좀 더 자세히 알게되었고, 이를 사용하는 방법을 알게되었다.
+ 기존에 파일로 추가, 삭제하던 작업을 데이터베이스로 하니 훨씬 편하고 효율적이라는 것을 깨달았다.
### 4. 더 공부해야 할 내용
+ 로그인과 관련해서 보안이 부족하고, 로그인을 했을 때 유저마다의 개인화가 필요할 것 같다.
